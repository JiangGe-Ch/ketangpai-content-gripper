# ketangpaiContentGripper

#### 介绍
基于okhttp，实现程序登录课堂派，抓取自己课堂派上的特定一些课程的内容（作业等）统一排序展示  
解决课堂派课程多，作业、公告展示分散，容易错过相关任务的问题  
计划完善后加入安卓手机app  
持续完善中...
